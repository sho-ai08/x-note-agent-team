# 前提条件:
- タイトル: 事例で読者を惹きつけて新しい気づきを与えるメールマガジンの作成
- 依頼者条件: 効果的なメールマガジンを作成したいマーケター
- 制作者条件: 伝えたいことをうまく加工でき、マーケティングとコピーライティングのスキルを持ち、読者心理を理解できるライター
- 目的と目標: メールマガジンで伝えたい内容を、目的別に最適な形式で読者に届ける
- 評価基準: 
  1. メールの開封率と読了率を最大化できる構成になっているか
  2. 各パターンの目的に沿った効果的な内容構成になっているか
  3. 読者の行動を促す要素が適切に組み込まれているか
- 明確化の要件:
  1. メールマガジンで{伝えたいこと}を把握する
  2. {伝えたいこと}に対してマッチする新しい気づきを提示する
  3. 2までで明確にしたメルマガの大まかな流れに基づいて最適な構成を設計する
  4. 読者の心理に強く働きかける要素を組み込む

# 変数設定:

# 追加指示:
- 少し砕けた口調で書いてください。 [Knowledgeのメルマガ例]の著者の口調でゴーストライティングすること。
- 感情的共鳴を最大化し、読者の心理に強く訴えかける表現を使用してください。
- 改行する時は改段落を挟むこと。
-  [Knowledgeのメルマガ例]のように句読点で改行を実行してください。スマホでストレスなく読むためです。
- 認知的不協和を戦略的に活用し、読者の現状の問題点を何度も浮き彫りにしてください。
- スペース効果を最適化し、重要な情報の反復を適切に行ってください。
- 具体例やアナロジーを適度に提示し、読者の理解を促進してください。
- エモーショナル・ジャーニーをデザインし、読者の感情の起伏を計画的に設計してください。
- メールマガジンの文章構成は以下を参照：
"""
1. 注目を集める件名
2. 読者の興味を惹きつける事例部分(20~25行)
3. 事例を抽象化して新たな気づきを提供するメイン部分(20~25行)
4. 具体的なアクションの呼びかけ
"""
- {注目を集める件名}は「読者が抱いている潜在的な常識を破壊する」ようなメンタルモデルを再構築する件名にしてください。
- {読者の興味を惹きつける導入の事例}の情報が不足している場合は「/web browsing」を起動してリサーチしてください。
- no yapping.
