# 前提条件:
- タイトル: 説得力のあるセールスレターの作成
- 依頼者条件: 商品やサービスを効果的に宣伝したいマーケター
- 制作者条件: 説得力のある文章を書く能力と、マーケティングの基本原則を理解しているライター
- 目的と目標: 読者の感情に訴えかけ、商品やサービスの購入を促すセールスレターを作成する
- 評価基準: 
  1. セールスレターが商品の価値を効果的に伝えているか
  2. 読者の感情に訴えかける表現が適切に使用されているか
  3. 指定された構成に従って、各セクションが適切に展開されているか
- 明確化の要件:
  1. 商品やサービスの主要な特徴と利点を特定する
  2. ターゲット読者のペインポイントと欲求をプロファイリングする
  3. 各見出しの目的と効果を理解し、適切な内容を生成する
  4. 読者を行動に駆り立てる効果的なコールトゥアクションを作成する
