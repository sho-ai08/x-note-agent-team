# バリューラダー構築プロンプト

## STEP1:
今から私の[プロフィール]（ターゲット層、ブランドコンセプトを含む）と[販売商品一覧]（商品名・価格・特徴など）を送ります。その内容を分析し、価格帯・ターゲットに応じたバリューラダーを表形式で設計してください。売上最大化を目的とし、顧客のLTV向上を考慮した構成としてください。各商品の特徴（価格・機能・ベネフィット・競合比較）をプロファイリングし、セールスシーケンスに沿ったバリューラダーを設計してください。バリューラダーの最初には、顧客を惹きつけるリードマグネット（例：無料PDF、ウェビナー、無料サンプル）を配置してください。なお、エントリー商品が欲しくさせるリードマグネットを提案すること。商品数に応じて、バリューラダーのサイクルを増加させてください(バリューラダーの順序の決定理由は解説すること)。バリューラダー1セットは2商品間とします。いいですね？

## STEP2:
構築されたバリューラダーを受け取ってください。そして、商品①から商品②へと顧客を移行させるための30日間(30通)のステップメールを設計してください。30日間(30通)のステップメールでは、商品②のセールスにつなげるために、価値提供（教育）を行ってください。30日のステップメール内ではセールスは不要です。メールのタイトルと内容を必ず**1日ずつ**提案して表形式にまとめてください。バリューラダーの各サイクルごとに設計してください。

# 制約条件:
- 各STEPはチャットを分割すること。
