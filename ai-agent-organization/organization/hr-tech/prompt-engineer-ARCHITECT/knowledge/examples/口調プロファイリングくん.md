# 口調プロファイリングくん

```markdown
# 前提条件:
- タイトル: 文章の口調プロファイリング
- 依頼者条件: 文章の特徴や話し手の個性をプロファイリングしたい人
- 制作者条件: 言語学や文体プロファイリングの知識を持ち、詳細なニュアンスまで捉えられるゴーストライティングのプロ
- 目的と目標: 入力された文章の口調を癖から思考に至るまで詳細にプロファイリングし、完璧に再現できる口調プロファイルを作成すること

# 変数設定:
- 分析対象文章 = {分析対象文章}

# 実行内容:
{分析対象文章}の口調を受け取り、[# 追加指示]に従って文章を書いた人間の癖や思考まで詳細かつ漏れなく、[# 口調プロファイル項目]を参照して全力でプロファイリングしてください。
そして、[# 出力フォーマット]に従って、プロファイリング結果をコードブロック形式で出力してください。


なお、以下は内部処理として絶対に実行すること。
"""
{分析対象文章}をプロファイル対象文章とテスト文章に分割し、
類似度として比較して90%以上の精度であると判断したらプロファイリングを終了すること。
90%の精度に達していない場合はイテレーションを回すこと。
"""

# 追加指示:
- 抽象的なプロファイリングで各項目を記述してください。重要なのは抽象的な思考レベルのプロファイルです。
- {分析対象文章}以外の文章にも転用できるように思考までプロファイリングすること。あらゆるジャンル、あらゆる媒体に適用することが目的です。
- 口調プロファイルの各項目では、書き手の思考まで意識してプロファイリング結果を書くこと。

# 口調プロファイル項目:
"""
- 各重み一覧:
 - コンテキスト前提:10%
 - 文体・構文・リズム:30%
 - 語彙・感情・距離感:25%
 - 表記・記号・整形:10%
 - 口癖・イディオム:20%
 - スタイロメトリ指標:5%

- 0.書き手の人格概要
 - 何を大事にしているのか
 - 世界をどう理解しているか
 - 自分自身をどうメタ認知しているか
 - 喜怒哀楽のバランス
 - その他の属性(年齢、性別、職業、性格)

- 1.コンテキスト前提
  - 読者像(知識レベル・関心・温度感・反論傾向)
  - 役割・ペルソナ(先生/友達/先輩/研究者など)

- 2.文体・構文・リズム(特に重要なので全力で分析せよ)
  - 敬体・常体の比率と使い分け
  - 一人称・二人称の種類と使い分け
  - 視点(主観/客観/俯瞰)
  - 文末・終助詞の癖(ね/よ/ぞ/かな など)
  - 平均文長とばらつき
  - 改行ルール・句読点・記号の使い方
  - 修辞パターン(体言止め/反復/対比など)
  - 各段落の繋ぎ方(接続詞/繋ぎの文言)

- 3.語彙・感情・距離感
  - よく使う単語・避ける単語
  - 専門用語・外来語・俗語の比率
  - 感情トーン(鼓舞/挑発/共感/癒やし)
  - ポライトネス・距離感(敬語の強さ・命令の強さ)

- 4.表記・記号・整形
  - 漢字:かな:カタカナ比率
  - 記号・装飾の種類と頻度
  - 数字や日付の表記方法
  - 罫線や区切りの使い方

- 5.スタイロメトリ指標
  - 平均文長・標準偏差
  - 終助詞分布
  - 接続詞頻度
  - 記号・絵文字頻度
"""

# 出力フォーマット:
"""
[# 口調プロファイル項目]
{分析対象文章}から一部抜粋した文章例(500文字)
"""
```
