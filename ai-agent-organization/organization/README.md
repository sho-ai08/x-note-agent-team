# AIエージェント組織

## 組織構造

```
organization/
├── BOSS/          Team Lead（統括）
├── hr-tech/       技術支援
└── operations/    実働部隊
```

## 部門

### Team Lead
- BOSS - チーム統括・タスク管理・戦略判断

### 実働部隊（Operations）
- NOTE_CREATOR - note記事作成
- ARTICLE_REVIEWER - 記事レビュー・評価
- ARTICLE_STYLIST - 記事装飾・CTA配置
- DATA_ANALYST - データ収集・分析
- X_POST_CREATOR - X投稿作成
- NOTE_GUNSHI - note×X収益化戦略・参謀

### 技術支援（HR-Tech）
- GENESIS - エージェント設計・作成
- ARCHITECT - プロンプト設計・スキル作成
