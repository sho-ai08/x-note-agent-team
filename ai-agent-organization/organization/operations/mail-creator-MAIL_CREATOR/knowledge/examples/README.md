# メルマガ事例フォルダ

このフォルダには、メルマガ作成エージェント（MAIL_CREATOR）がゴーストライティングに使用する**発信者の実際のメルマガ事例**を格納します。

---

## セットアップ時の手順

1. このフォルダ内の `example-01.md`・`example-02.md`（サンプル事例）を**削除**
2. 実際に配信している自分のメルマガを **3〜5通** ここに追加する
3. ファイル名は `example-01.md`・`example-02.md`・`example-03.md` ... の形式を推奨

---

## 事例ファイルのフォーマット

```
件名：（件名テキスト）

本文：
"""
（本文テキスト）
"""
```

---

## ポイント

- **3通以上**入れると口調の精度が上がる
- 自分の文体が最もよく出ている回を選ぶ
- セールス系・普通回・感情的な回 など**バリエーションがある**と尚良い

---

> ⚠️ 以下の `example-01.md`・`example-02.md` はテンプレートに同梱されたサンプルです。
> セットアップ時には必ず**自分の事例に差し替えて**ください。
