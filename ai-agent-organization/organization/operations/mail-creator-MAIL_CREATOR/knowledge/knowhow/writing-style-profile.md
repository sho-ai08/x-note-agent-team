# 発信者 文体プロファイル

> **⚠️ セットアップ時の注意**
> このファイルはテンプレートです。
> 実際に使用する前に、発信者の文体に合わせてすべての項目を埋めてください。
> 埋めた内容は MAIL_CREATOR エージェントがメルマガ作成時に参照します。

---

## 基本情報

| 項目 | 内容 |
|------|------|
| 発信者名 | （例: しょー） |
| 一人称 | （例: 僕） |
| 読者への呼びかけ | （例: あなた、みなさん） |
| ニックネーム/肩書き | （例: 副業で1,000万稼いだ大学院生） |

## 口調・文体の特徴

### トーン
- （例: 少し砕けた口調。友達に話しかけるような感覚）
- （例: 「笑」「...」を多用してテンポを作る）
- （例: 敬語ベースだが、固くなりすぎない）

### 文末パターン（よく使うもの）
- （例: 〜わけです、〜なんですね、〜ですよね...）
- （例: 「笑」を文末や行末に入れる）
- （例: 「...」で余韻を作る）

### 改行・段落の特徴
- （例: 句読点ごとに改行する）
- （例: 1文1行を基本とする）
- （例: 空行（スペース）を多用してリズムを作る）

---

## 文章構造の特徴

### 導入パターン
- （例: 近況・雑談から入る。「こんばんは、〇〇です。」で始まる）
- （例: 「余談はここまでにして...」で本題へ）

### 強調方法
- （例: ━━━━━━━━━━ で囲む）
- （例: 「」で重要フレーズを括る）
- （例: ✅や✔︎を使った箇条書き）

### 締め方のパターン
- （例: 「それでは。」で締める）
- （例: 読者へのアクション呼びかけで終わる）

## 頻出表現・口癖

```
（例）
- 「〜わけです」
- 「〜なんですね」
- 「笑」
- 「まじで」
- 「ビビります」
- 「エグい」
```

## NG表現（この発信者が使わない表現）

```
（例）
- 「非常に」「極めて」などの堅い表現
- 「〜いたします」などの過度な敬語
- 「本日は〜についてお伝えします」などの形式的な書き出し
```
