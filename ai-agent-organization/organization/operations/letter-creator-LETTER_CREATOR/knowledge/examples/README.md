# レター事例ディレクトリ

このディレクトリには、発信者が実際に作成・使用したセールスレターの事例を格納します。

## 用途

LETTER_CREATOR エージェントがレター作成時に参照します。
**構成の大まかな流れのみを参照してください。内容の学習や流用は禁止です。**

## 参照ポイント

- 各セクションのボリューム感（文字数・段落数）
- 見出しの流れと順序
- 読者への語りかけ方のトーン
- 改行・空行のリズム感
