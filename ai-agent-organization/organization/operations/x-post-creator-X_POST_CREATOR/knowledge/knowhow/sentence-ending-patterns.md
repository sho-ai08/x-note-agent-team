# 語尾バリエーションリスト

語尾の連続を防ぐためのナレッジです。**同じ語尾を2回連続で使わない**ことを鉄則とし、このリストを参照しながら執筆してください。

---

## 使い方

1. 1段落書くごとに、直前の文の語尾と被っていないか確認する
2. 被っていたら、このリストから別カテゴリの語尾に差し替える
3. 3文以上書いたら、語尾のカテゴリが3つ以上に分散しているか確認する

---

## カテゴリ別 語尾リスト

### A. 断定系（言い切り）
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜です | ニュートラルな断定 | これがポイントです |
| 〜なわけです | 論理的な帰結 | だからこそ大事なわけです |
| 〜なんです | 強調を込めた断定 | ここが勝負を分けるんです |
| 〜でして | 柔らかい断定 | これがめちゃくちゃ重要でして |
| 〜なんですよ | 会話的な断定 | 実はこれがカギなんですよ |
| 〜に尽きる | 強い断言 | 結局、準備に尽きる |
| 〜ってことです | 砕けた断定 | つまりそういうことってことです |

### B. 体言止め系
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜ということ。 | 余韻を残す体言止め | すべてを左右するということ。 |
| 〜というわけ。 | 結論の体言止め | だから必要というわけ。 |
| 〜だけ。 | ミニマルな体言止め | やることはたったこれだけ。 |
| 〜のか。 | 問いかけ的な体言止め | 本当にそれでいいのか。 |

### C. 問いかけ系
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜ですよね？ | 共感を誘う | 経験ありますよね？ |
| 〜だと思いません？ | 読者に考えさせる | これ、おかしいと思いません？ |
| 〜じゃないですか？ | カジュアルな共感 | 誰でもそうなりますよねじゃないですか？ |
| 〜ありませんか。 | 体験を問う | こんな経験ありませんか。 |

### D. 余韻・推量系
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜なんですけどね。 | 独り言っぽい余韻 | いや、これがおもろいんですけどね。 |
| 〜だったりします。 | 柔らかい提示 | 意外とこれが効いたりします。 |
| 〜かもしれません。 | 控えめな推量 | 今年一番の発見かもしれません。 |
| 〜と思ってます。 | 個人的見解 | これが最適解だと思ってます。 |
| 〜気がしてます。 | さらに控えめ | なんかそんな気がしてます。 |

### E. フランク系
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜なんですよね。 | 親しみのある共感 | ここが難しいんですよね。 |
| 〜ってこと。 | 砕けたまとめ | 要はそういうことってこと。 |
| 〜わけですよ。 | 説明的フランク | だからこうなるわけですよ。 |
| 〜してるんです。 | 体験ベース | 僕もずっとそうしてるんです。 |

### F. 変化球系（スパイス用）
| 語尾 | ニュアンス | 使用例 |
|------|-----------|--------|
| 〜してしまうんです。 | 意外性・残念 | 全部忘れてしまうんです。 |
| 〜せざるを得なくなる。 | 必然性 | 処理せざるを得なくなる。 |
| 〜仕組みになってます。 | 構造的説明 | そういう仕組みになってます。 |
| 〜に変わりました。 | 変化の報告 | そこからガラッとに変わりました。 |
| 〜ところがある。 | 客観的指摘 | ここが見落とされがちなところがある。 |
| 〜してほしい。 | 直接的お願い | ぜひ試してほしい。 |

## 語尾ローテーション例

以下は5文連続で語尾を被らせない書き方の見本です:

```
これがめちゃくちゃ重要でして。（E:断定）
なぜかというと、AIは文脈を理解する生き物なんですよね。（E:フランク）
だからこそ、プロンプトの構造が勝負を分けるわけです。（A:断定）
ここを見落としてる人が本当に多い。（B:体言止め）
あなたはどうですか？（C:問いかけ）
```

---

## チェック方法

執筆後に以下を確認:
1. 同じ語尾が2回連続していないか
2. 同じカテゴリ（A〜F）が3回連続していないか
3. 全体を通して3カテゴリ以上が使われているか
4. 「なんですよね」は**記事全体で5〜8回程度**が自然（最頻出語尾。制限しすぎると逆に不自然になる）
