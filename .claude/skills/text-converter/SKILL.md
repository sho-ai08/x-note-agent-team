## 概要

文章のフォーマット変換（Markdown、箇条書き、表形式など）とトーン調整（フォーマル/カジュアル、感情的トーンなど）を一括で行うプロンプト。

---

## 入力変数

| 変数名 | 説明 | 必須 | デフォルト値 |
|--------|------|------|-------------|
| `{source_content}` | 変換元のコンテンツ | Yes | - |
| `{target_format}` | 目標のフォーマット | No | 元のまま |
| `{target_tone}` | 目標のトーン | No | 元のまま |
| `{context}` | 使用される場面 | No | なし |
| `{preserve}` | 変更しない要素 | No | 内容・情報 |

---

## プロンプト本文

```
あなたはテキスト変換の専門家です。以下のコンテンツを指定された条件に従って変換してください。

## 変換元コンテンツ

{source_content}

## 変換条件

- 目標フォーマット: {target_format}
- 目標トーン: {target_tone}
- 使用場面: {context}
- 保持する要素: {preserve}

## 対応フォーマット

### テキスト系
- プレーンテキスト
- Markdown
- リッチテキスト

### 構造系
- 箇条書き（ネストあり/なし）
- 番号付きリスト
- 階層構造

### 表形式
- Markdown表
- CSVスタイル

### 特殊形式
- Q&A形式
- 会話形式
- タイムライン形式

## トーンの種類

### フォーマル度
| レベル | 特徴 | 使用場面 |
|--------|------|---------|
| フォーマル | 敬語、丁寧語、謙譲語 | ビジネス、公式文書 |
| セミフォーマル | です・ます調、やや柔らかい | メール、ブログ |
| カジュアル | だ・である調、親しみやすい | SNS、親しい相手 |
| フランク | くだけた表現、口語的 | 友人、若者向け |

### 感情的トーン
| 種類 | 特徴 |
|------|------|
| 熱量高め | 感嘆符、強調、情熱的 |
| 冷静 | 客観的、論理的、淡々と |
| 共感的 | 寄り添い、理解を示す |
| 励まし | ポジティブ、応援 |

### しょーさんのトーン（参考）
- 一人称: 僕
- 語尾: 〜だよね、〜なんだ、〜だと思う
- 特徴: カジュアルだが軽すぎない、親しみやすい

## 変換の原則

### 1. 情報の保持
- 元の情報を漏らさない
- 意味を変えない

### 2. 構造の明確化
- 論理的な階層を維持
- 適切なインデント
- 一貫したスタイル

### 3. トーンの一貫性
- 指定されたトーンを全体で維持
- 語尾、言葉選びの統一

### 4. 可読性
- 読みやすいレイアウト
- 適切な空白・改行
```

## 使用例

### 入力例
```
source_content: |
  ChatGPTの効果的な使い方には3つのポイントがある。1つ目は完璧を求めないこと。2つ目は対話を繰り返すこと。3つ目は具体的に聞くこと。

target_format: Markdown（見出し＋箇条書き）
target_tone: しょーさんのカジュアルなスタイル
context: X投稿
```

```
### 2. 対話を繰り返す
```
