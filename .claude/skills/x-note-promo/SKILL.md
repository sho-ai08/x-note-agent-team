# note記事紹介投稿作成

# 前提条件:
- タイトル: note記事を紹介するX投稿を作成する
- 依頼者条件: noteで公開した記事をXで宣伝したい情報発信者
- 制作者条件: 高度なライティングスキルを持ち、人々を惹きつける文章を書けるライター
- 目的と目標: note記事の魅力を長文で分かりやすくフォロワーに伝える。そして、フォロワーにnote記事を読んでもらう。
- 実施要件:
  1. 入力された{note記事タイトル}と{note記事要約}の内容をプロファイリングし、主要なポイントや関連情報を特定する
  2. 読者を惹きつけるリード文でX投稿を開始する
  3. note記事を読みたくなるような訴求を並べる
  4. CTAポイントへと誘導する

# 変数設定:
- note記事タイトル = {note記事タイトル}
- note記事要約 = {note記事要約}

# 実行内容:
{note記事タイトル}のnote記事を紹介します。
{note記事要約}を参照してから、
[フォーマット例]の書き方を参考にして、
note記事紹介用のX投稿を書いてください。

# 追加指示:
- 文字制限は撤廃してください。
- 感情的共鳴を最大化し、読者の心理に強く訴えかける表現を使用してください。
- 認知的不協和を戦略的に活用し、現状の問題点を浮き彫りにしてください。
- エモーショナル・ジャーニーをデザインし、読者の感情の起伏を計画的に設計してください。
- パラグラフ構造を最適化し、読みやすさを向上させてください。
- 複数レベルのターゲット層に刺さるようにしてください。
- note記事を読むことで手に入れられる理想の未来を複数の切り口から書いてください。
- note記事を読まないことによる機会損失を複数の切り口から書いてください。
- 読者がイメージしやすくなるように数字を多用して具体性を強めてください。
- CTAポイントは「続きはnoteで」「詳しくはnoteに書きました」などnote記事への誘導にしてください。
- [フォーマット例]からは**文章構成だけ**を抽出してください。
- 少し砕けた口調で書いてください。

# 重要な制約（必ず守ること）:
- **🔴 1行目フックが全て**: 1行目はターゲットが普段から目にしている言葉・日常の心の声をそのまま置く。「自分のことだ」と思わせる一文で始める
  - 気づき・共感系: 読者が心の中で言っているネガティブな言葉や自問をそのまま1行目に（例: 「AIを勉強しなきゃ」と焦って動画を見まくる。）
  - ノウハウ系: 読者が手に入る理想の未来・ベネフィットを具体的に1行目で提示（例: ○○するだけで、◯◯が△△になった話。）
- **穴を広げる構造**: 1行目でグサッと刺したら、同じテーマで「あるある」「共感」「気づき」を重ねて穴を広げる。読者が「これ自分のことだ」と深くなったところで提言→「それをnoteに書きました」で締める
- **じらして誘導**: 結論・理由・まとめは書かない。「続きはnoteで」の動機を作るのが目的
- **改行しすぎない**: 一文ごとに改行するのはNG。数行の塊で訴求するイメージで書く
- **自然体のトーン**: 「〜なんですよね」「〜だったりします」など押し付けがましくない表現を使う
- **興味喚起を入れる**: 「じゃあどうやって？」「それがあるんです」のように読者が気になる形に
- **AI臭さ排除**: 安全クッション（「一般的に」「一概には」）は削除。抽象語（「重要」「効果的」）だけで押し切らず具体的に書く。同義語連打（重要・大切・欠かせない）はせず1回で言い切る。締め定型句（「参考になれば幸いです」等）は使わない
