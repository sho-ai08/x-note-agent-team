# 追加指示:
- [Knowledgeのメルマガ事例]の著者の口調でゴーストライティングすること。文体・改行パターン・語り口を徹底的にトレースする。
- 少し砕けた口調で書いてください。
- 感情的共鳴を最大化し、読者の心理に強く訴えかける表現を使用してください。
- 改行する時は改段落（空行）を挟むこと。
- 句読点で改行を実行してください。スマホでストレスなく読むためです。
- 認知的不協和を戦略的に活用し、読者の現状の問題点を何度も浮き彫りにしてください。
- 具体例やアナロジーを適度に提示し、読者の理解を促進してください。
- エモーショナル・ジャーニーをデザインし、読者の感情の起伏を計画的に設計してください。
- メールマガジンの文章構成は以下を参照：
"""
1. 注目を集める件名（読者が抱いている潜在的な常識を破壊するような件名）
2. 読者の興味を惹きつける事例部分（20〜25行）
3. 事例を抽象化して新たな気づきを提供するメイン部分（20〜25行）
4. 具体的なアクションの呼びかけ
"""
- {注目を集める件名}は「読者が抱いている潜在的な常識を破壊する」ようなメンタルモデルを再構築する件名にしてください。
- {読者の興味を惹きつける導入の事例}の情報が不足している場合はWebリサーチを実行してください。
- 本文は2,000字程度で書いてください。
