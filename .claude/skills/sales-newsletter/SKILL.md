# 入力:
- セールスレター（商品情報） = {セールスレター}
- 販売開始日時 = {販売開始日時}
- 販売終了日時 = {販売終了日時}

# 変数設定（セールスレターから抽出）:
- 商品名 = {商品名}
- 通常価格 = {通常価格}
- 特別価格 = {特別価格}
- 商品特徴 = {商品特徴}
- ベネフィット = {ベネフィット}
- 成功事例 = {成功事例}
- FAQ = {FAQ}

# 前提条件:
- タイトル: 期間限定セールスメールシナリオの作成
- 依頼者条件: 期間限定セールで効果的なメール配信を行いたいマーケター
- 制作者条件: セールスコピーライティングとメール配信の知識を持つプロフェッショナル
- 目的と目標: 期限を活用して段階的な情報提供と心理的緊張感を醸成し、購買行動を促進する

# 実行内容:
上記の情報をもとに、{商品名}の3日間限定セールに関する9通のメールシナリオを作成する。
以下の[追加指示]および[フォーマット]に従うこと。

# 実施要件:
1. セールスレターから必要な情報を適切に抽出して[変数設定]に格納すること。
2. 合計9通のステップメールを、時系列に沿って作成すること。
3. 各メールは1通ずつ作成し、その目的と狙いが明確に伝わるようにすること。
4. メール本文内で、期間限定の特別価格・残り時間など「期限訴求」を効果的に組み込むこと。
5. 購入を躊躇する読者の心理的障壁を段階的に解消する構成にすること。

# 追加指示:

## 全体の追加指示:
- {販売開始日時}と{販売終了日時}は必ず守ること。
- 残り時間訴求は3日目のみ行うこと。
- メールは1通ずつ提示すること。1通作成ごとに「次」という指示を待つこと。
- 限定価格を強調し、読者に行動を促す表現を適宜使用すること。
- 各メール配信時刻に応じて、徐々に緊迫感を高めていくストーリー構成とすること。
- 感情に訴えかける表現を効果的に使用する。
- 読みやすい文章構成を維持する（改行・空行を多用してスマホ最適化）。
- コピー文は少し砕けた口調で書いてください。
- 本文は2,000字程度で書いてください。
- 販売開始までは価格は絶対に非公開にしてください。1通目・2通目は価格の公開を禁止とします。
- 1通目から4通目までは同日（1日目）である。

## 9通のメールシナリオ構成:
"""
・1日目 昼（12:00）　販売開始前の期待感醸成
・1日目 夜①（18:00）販売1時間前の告知
・1日目 夜②（19:00）販売開始の案内
・1日目 夜③（20:00）販売開始1時間後のフォロー
・2日目 昼（12:00）　成功事例訴求
・2日目 夜（19:00）　FAQ提示・不安解消
・3日目 昼（12:00）　12時間前の告知
・3日目 夜①（18:00）残り数時間の緊迫感
・3日目 夜②（23:30）最終告知（残り時間30分）
"""

## 各日の追加指示:

### 1日目の追加指示:
- 期待感とワクワク感を醸成する
- 特別価格の訴求を明確に行う（ただし1通目・2通目は価格非公開）
- 即時行動を促す要素を含める

### 2日目の追加指示:
- 成功事例で説得力を高める
- FAQで不安を解消する
- 残り時間を意識させる

### 3日目の追加指示:
- 緊迫感を最大限に高める
- 「今しかない」という感覚を強調する
- 具体的な残り時間を示す

# フォーマット:
"""
◆[通数] [日・時間帯]（[目的]）

【件名】
（件名を明記）

【本文】
（本文約2,000字程度）
（最後にフッターを挿入）
"""

# 絶対条件:
- 1通作成するごとに出力を停止し、「次」の入力を待つこと。
- 価格は1通目・2通目では絶対に記載しないこと。
- AI臭い表現（定型的な言い回し、感情のない説明文）は使用禁止。
- 各メールに明確な「感情の目的」を持たせること（期待・興奮・安心・焦り・決断）。
