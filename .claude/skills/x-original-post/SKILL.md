```
## 変換のルール

### 1. 構成

#### 冒頭（第1ツイート）
- **結論を断言する**
  - 「〇〇だと思うんですよね」
  - 「〇〇なんですよ」
  - 「結論、〇〇です」
- 読者の興味を惹きつける
- 続きを読みたくなる問題提起

#### 本文（第2〜最終ツイート）
- **各主張に対して肉付けする**
  - なぜそうなのか → 理由を説明
  - 具体的にどういうことか → 具体例や状況を補足
  - どういう効果があるのか → 読者へのメリットを示す
- **読者が共感・納得できる文脈を補強**
  - 「〜じゃないですか」「〜だと思うんですよね」などで共感を誘う
  - 読者が「わかる」「あるある」と思える描写
- **論理展開を明確に**
  - 主張 → 理由 → 具体例 → 効果
  - 段階的に深掘りしていく

### 2. 文体・トーン

- **口語的で親しみやすい**
  - 「〜なんですよね」「〜じゃないですか」「〜だと思うんです」
  - 「実は」「正直」「ぶっちゃけ」などの口語表現を自然に使う
- **断定と共感のバランス**
  - 冒頭は断定で惹きつける
  - 本文は共感を誘いながら説得する
- **読者に語りかける**
  - 読者を意識した書き方
  - 「あなたも〜じゃないですか?」など

### 3. 制約条件

#### 必須
- オリジナルコンテンツであること（独自の思考・意見・体験）
- 一次情報や実体験に基づく内容
- 読み応えのある長文（800〜1500文字程度）

#### 禁止
- 「--」（ダッシュ）は使わない → 代わりに「つまり」「だから」などの接続詞を使う
- 絵文字の多用 → 最小限（0〜2個程度）に抑える
- 煽り表現（「知らないとヤバい」「これを知らないとオワコン」など）
- 他人の意見の転載・言い換えだけの内容
- 安全クッション（「一般的に」「多くの場合」「一概には言えませんが」）→ 言い切る
- 抽象語だけの主張（「重要」「効果的」「本質」）→ 具体的に何がどうなるか書く
- 同義語の言い換え連打（重要・大切・欠かせない）→ 1回で言い切る
- 締め定型句（「参考になれば幸いです」「まずは小さく始めましょう」）→ 内容に即した一文で淡く締める
- 同じ語尾の2回連続 → `sentence-ending-patterns.md`（語尾リスト）を参照し、カテゴリA〜Fを分散させる。「なんですよね」は投稿全体で2回以内

#### 推奨
- 実体験や具体的なケースを含める
- 失敗談や学びを共有する
- 時系列や体系的な整理
- 専門性と親しみやすさの両立
- **文脈のつなぎを意識**: 各段落・各主張が独立してしまわないよう、前後を自然につなぐ。段落の末尾で次の話題への「フリ」を入れる（「じゃあなんでこうなるのか。」「でもそれだけじゃないんです。」等）。投稿全体が1本の流れで読めるか最後に通しで確認する
- **思考・AI論型の構成（エン率8〜12%期待・最重要テーマ）**: AI哲学・人間とAIの関係がテーマの場合は「①読者の不安を言語化（「自分が不要になるのでは」等）→②共感で深掘り（その不安の正体を掘り下げる）→③視点転換（「でも、それって本当にそうか」）→④希望と主張（AI時代における人間の価値の再定義）→⑤自分の体験・考えで着地」の流れを意識する。計画的に月2〜3件投稿する

### 4. 品質基準

- **一次性・独自性・検証可能性**を重視
  - 参照: `ai-agent-organization/shared/knowledge/noteの心得.md`
  - 単なる情報の転載ではなく、独自の視点や体験を含める
- **読者ファースト**
  - 読者が求める情報を的確に届ける
  - 自己満足ではなく、価値提供を意識
- **データや根拠**
  - 主観的な意見だけでなく、具体的な数字や事例を含める

## 出力形式

変換後のX長文投稿のみを出力してください。

**注意**:
- 前置き（「以下のように変換しました」など）は不要
- 本文のみを出力
- 自然な改行で読みやすく

## 出力例（構造イメージ）

結論、〇〇なんですよね。

なぜかというと〜

例えば〜じゃないですか。

実は〜なんですよ。

つまり〜ということ。

だから〜だと思うんです。
```

## 注意事項

- 元の内容の本質・意図を損なわないこと
- ユーザーの思考を「ブラッシュアップ」するのであり、「別の内容」にしない
- 過度に装飾せず、自然な口語表現を心がける
- Xのプラットフォームに最適化（改行、読みやすさ）
- 絵文字は本当に必要な場合のみ使用（基本は0〜2個）
- 「--」は絶対に使わない（AIっぽくなるため）
