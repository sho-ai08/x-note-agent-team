# 文字起こし整形プロンプト

## 概要
動画の文字起こしを整形するユーティリティプロンプト。
組織内の誰でも使用可能な汎用ツール。

---

## 用途
- 動画コンテンツの文章化
- インタビュー音声の整形
- セミナー・講演の書き起こし整形
- ポッドキャスト内容の整形

## 使用可能エージェント
- NOTE_CREATOR: 元情報として活用
- 全エージェント: 汎用ツールとして使用可能

---

# 文字起こし整形テンプレート

## 入力:
- 文字起こし = {文字起こし}

## 前提条件:
- タスク: 動画の文字起こしを整形する
- 依頼者条件: 動画コンテンツを文章化したいコンテンツクリエイター
- 制作者条件: 動画コンテンツを文章にできるライティングスキルを持つ人
- 目的と目標: 動画の文字起こしを読みやすく整形する

## 実行内容:
添付した{文字起こし}の内容は省略せず、
以下の追加指示に従ってすべて書き起こしてください。

## 追加指示:
- {文字起こし}の内容は省略せずすべて書き起こしてください
- 要約は禁止とする
- 1チャットに書ききれない場合はユーザからの「続き」という入力を待ってください
- 適切なタイミングで強制的に「句読点」や「クエスチョンマーク」を入れてください
- スマートフォンで読みやすいように改行や改段落を最適化してください

## フォーマットルール:
以下のポイントに強制的に従ってください。

```
①1文ごとに改行を実行
②2文,1文,3文と改段落位置のリズムを最適化
```

## ノイズ除去:
以下の「意味をなさないノイズや相槌」やその他の「考える間」などのノイズは強制的に除去してください。

```
①えー
②あー
③うん
④ま
⑤え
⑥うんうん
⑦ね
⑧うん、うん
⑨なんか
```

## 追加処理:
- 文字起こしが適切にできていない箇所は、前後の文脈から推論してください

---

## 出力後の次のステップ

整形した文字起こしは、以下の用途で活用できます:

1. **note記事作成**: `note-creation` スキルで記事化
2. **構成作成**: `article-structure` スキルで見出し構成を設計
3. **マインドマップ化**: 階層的に整理する場合は別途プロンプトを使用
